Frequently Asked Questions

1. What are "StockBites"?

A StockBite represents an anonymous stock prediction. It has four key components:

  • An associated security
  • A message
  • An anoynmous author
  • A date

2. What is the origin of these anonymous predictions?

The StockBite predictions are gathered using various social media APIs and web-scraping libraries. The list of StockBite sources includes:

  • Twitter
  • StockTwits
  • Online message boards
  • Stock forums / Reddit

3. How accurately can sentiment measure a security's movement?

Sentiment analysis is merely one of a host of tools that should be used to understand and predict the global stock market. StockBite's anonymous sentiment is merely an indicator of the web's predictions for a security.
